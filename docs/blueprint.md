# **App Name**: Comic Cloud Reader

## Core Features:

- Comic Reader: Integrated reader that supports CBZ, CBR and PDF comic formats.
- Library Organization: Comic book collections organized by folder.
- Advanced Search: Searchable and filterable listings by title, author, series and user-defined tags.
- User Authentication: Account creation using Firebase authentication with email, Google, or Apple.
- Cover Generation: Thumbnail previews, or dynamic cover generation with an AI tool based on the series name or a description.
- Theme Toggle: Enable a light or dark theme for reading in different environments.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) evoking depth, mystery and sophistication, like classic comic book stories.
- Background color: Very dark gray (#222222) - nearly desaturated indigo - for a dark theme.
- Accent color: Saturated purple (#800080) - analogous to indigo - for highlights, selections, and calls to action.
- Headline font: 'Belleza', a humanist sans-serif with personality and art/design styling.
- Body font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Simple line icons for navigation and actions.
- Clean, grid-based layout for displaying comic covers.
